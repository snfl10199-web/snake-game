# Snake Game (뱀 게임)

간단한 HTML5 + JavaScript 기반 뱀 게임입니다.

## 실행 방법

- 브라우저에서 `index.html` 파일을 더블클릭하거나 열면 게임이 실행됩니다.
- 로컬 간이 서버를 사용하려면 (권장, 일부 브라우저의 파일 보안 정책 회피):

	- Windows PowerShell (간단):

```powershell
# 현재 디렉터리에서 8000 포트로 서버 실행 (Python이 설치되어 있어야 함)
python -m http.server 8000
# 브라우저에서 http://localhost:8000 으로 접속하여 `index.html` 열기
```

## 조작 방법

- 키보드: 방향키(또는 WASD)로 이동, `R`키로 재시작
- 모바일/터치: 화면 하단의 버튼으로 조작

## 파일

- `index.html` : 게임 UI와 캔버스
- `style.css`  : 간단한 스타일
- `script.js`  : 게임 로직 (뱀 이동, 사과, 충돌, 점수)
